What is Jsco?
----------------

Jsco is an experimental digital currency that enables instant payments to
anyone, anywhere in the world. Jsco uses peer-to-peer technology to operate
with no central authority: managing transactions and issuing money are carried
out collectively by the network. Jsco Core is the name of open source
software which enables the use of this currency.

What such JSCO 
-------

JSCO this is a website which will have a large database of licensed programs - scripts with the support of the creators themselves. 
The site will support russian and english, developers will be paid by the same users who are interested in their developments, 
all this will happen in automatic mode that will not waste time.
Not all programs - scripts will be paid, the developer can choose which license type (Closed, Paid, Open) 
Also on the site will be various competitions where the winner will receive money, and his program will be open access, 
which will allow the site to be constantly filled, and also collects the initial audience of programmers.
We can say that this project is made to support not known, but good programmers where they will reveal themselves. 
Most of the sites that are online now selling scripts receive money for yourself, without the knowledge of the parties of 
the same developers, we are we will provide their work and money. 
The site itself will be similar in structure github, but with multiple changes for easy searching and filling.
